Introduction to HTML and CSS

HTML and CSS is the basic foundation of the web.

HyperText Markup Language, or HTML, is the universal language of the internet. Every time you visit a website, the siteâ€™s web server sends HTML to your browser, which then renders it as the web page you see on your screen. HTML is not a programming language, but a markup language meant to be interpreted by web browsers in graphical form. We will be learning about the basic HTML makeup of a webpage.

HTML elements are added to the content of a page to describe its structure. An element consists of the opening tag, closing tag, and content.
